<h1 align="center">LearnCam</h1>

<h2>
Ingresos y costos de LearnCam, una aplicación creada para visualizar, administrar y comparar estadísticas de la empresa LearnCam.
</h2>

<br><br>

<p>
Esta aplicación fue diseñada para la empresa de educación independiente LearnCam. Funciona 100% en Java y está diseñada como una aplicación de escritorio. La app puede generar diferentes reportes mensuales o completos según sea requerido, basándose en los datos almacenados en el sistema. Aunque este sistema no cuenta con una conexión a una base de datos, es escalable para futuras actualizaciones. La app proporciona gráficas de barras de costos, ganancias y pagos dentro de la empresa, y también cuenta con un sistema de login. Además, ofrece 2 temas de color para una mejor visibilidad.
</p>
